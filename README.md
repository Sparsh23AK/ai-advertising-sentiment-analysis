# AI in Advertising: Sentiment & Emotion Analysis of YouTube Comments

## 📌 Project Overview
This project analyzes audience reactions to YouTube advertisements using Natural Language Processing (NLP).  
The goal is to extract sentiment and emotional patterns from user comments and translate them into actionable business insights for advertising strategy.

---

## 🎯 Objective
- Analyze sentiment (positive, negative, neutral) in YouTube comments  
- Identify dominant emotions (joy, trust, anticipation, etc.)  
- Generate insights to improve advertising effectiveness  

---

## 📊 Dataset
- Source: YouTube comments (scraped from official brand channels)  
- Total Records: 1,148 comments  
- Sectors Covered: FMCG, Automotive, Technology, Banking/Fintech, Food/Dairy  

---

## ⚙️ Tools & Technologies
- Python  
- NLP Libraries: VADER Sentiment Analyzer, NRCLex  
- Data Processing: Pandas, Regex  
- Visualization: Matplotlib  

---

## 🔍 Methodology
1. Data Collection (YouTube scraping)  
2. Data Cleaning (removing emojis, duplicates, noise)  
3. Sentiment Analysis using VADER  
4. Emotion Detection using NRCLex  
5. Visualization of sentiment and emotion distribution  

---

## 📈 Key Insights
- Neutral sentiment dominates (~50%), indicating informational engagement  
- Positive sentiment (~39%) driven by joy, trust, and anticipation  
- Negative sentiment (~11%) linked to pricing, repetition, or realism issues  
- Technology and Automotive sectors show higher emotional engagement  
- FMCG and Banking sectors show trust-based but neutral-heavy responses  

---

## 💡 Business Impact
- Emotional storytelling is key to converting neutral users into active engagement  
- Different industries require different emotional strategies  
- AI-driven sentiment analysis enables data-backed marketing decisions

## 🚀 Future Improvements
